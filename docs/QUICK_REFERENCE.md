## 🎯 项目概述

**项目名称**: 随机目的地大转盘  
**当前版本**: v4.0.0  
**数据类型**: v1.4.0  
**项目描述**: 一个精美的旋转转盘应用，用于随机抽取旅游目的地，支持 AI 驱动的目的地介绍和层级抽签功能。

---

## ✨ 设计亮点

### 🎨 视觉设计
- **现代紫色渐变主题**: 从传统的灰色升级为充满活力的紫色渐变配色
- **玻璃态效果**: 使用毛玻璃和渐变背景营造现代感
- **精致的阴影和光晕**: 微妙的多层阴影效果
- **响应式设计**: 完美适配手机、平板和桌面设备

### ⚡ 交互动画
- **流畅的转盘旋转**: 使用 Motion 实现的贝塞尔曲线动画
- **微交互反馈**: 按钮悬停、点击的即时反馈
- **入场动画**: 获奖卡片的弹性和淡入效果
- **彩纸效果**: 获奖时的 confetti 动画

### 📱 用户体验
- **单手操作优化**: 移动端触摸友好
- **即时反馈**: 每个操作都有视觉反馈
- **清晰的视觉层次**: 焦点始终在转盘上

## 🎰 核心功能

### 1. 转盘抽签 ⭐⭐⭐⭐⭐
- **层级导航**: 省 → 市 → 县三级联动
- **预设列表**: 中国34个省级行政区
- **自定义列表**: 用户可添加自己的目的地
- **流畅动画**: 8圈旋转 + 随机圈数

### 2. AI 智能介绍 ⭐⭐⭐⭐
- **Gemini 3 Flash**: 实时生成目的地介绍
- **百科链接**: 一键跳转 Wikipedia
- **优雅降级**: 无 API Key 时显示默认文案

### 3. 收藏与历史 ⭐⭐⭐⭐
- **收藏夹**: 保存心仪的目的地
- **历史记录**: 追踪最近50次抽取
- **本地存储**: LocalStorage 持久化

### 4. 主题切换 ⭐⭐⭐⭐
- **跟随系统**: 自动检测系统偏好
- **手动切换**: 一键切换深色/浅色模式
- **平滑过渡**: 0.3秒渐变动画

### 5. PWA 支持 ⭐⭐⭐⭐⭐
- **可安装**: 添加到主屏幕
- **离线可用**: Service Worker 缓存
- **原生体验**: 独立的浏览器窗口

## 🔄 动效规范

### 转盘旋转
```javascript
{
  duration: 6.5,           // 6.5秒
  ease: [0.15, 0.85, 0.2, 1],  // 自定义缓动
  extraSpins: 8,           // 额外8圈
  pointerJitter: ±4°        // 指针抖动
}
```

### 卡片入场
```javascript
{
  duration: 0.5,
  type: "spring",
  opacity: 0 → 1,
  y: 20 → 0,
  scale: 0.95 → 1
}
```

### 按钮交互
```javascript
// 悬停
{ scale: 1.05, y: -2 }

// 点击
{ scale: 0.97, y: 1 }

// 过渡
{ duration: 0.3, ease: "easeInOut" }
```

## 🎯 设计原则

### 1. 简洁直观
- 用户打开即可理解操作方式
- 核心功能一目了然
- 减少认知负担

### 2. 视觉焦点
- 转盘始终是页面中心
- 其他元素辅助而非干扰
- 使用对比和层次引导视线

### 3. 即时反馈
- 每个交互都有视觉响应
- 加载状态清晰可见
- 错误信息友好易理解

### 4. 流畅体验
- 60fps 动画保证丝滑感
- 避免突兀的跳转
- 使用过渡而非切换
