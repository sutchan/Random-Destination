# 随机目的地大转盘 - UI/UX 原型设计规范 v1.0

## 📐 设计原则

1. **简洁直观**：用户打开即可理解如何操作
2. **视觉焦点**：转盘是页面核心，其他元素辅助
3. **流畅体验**：每个交互都有即时视觉反馈
4. **现代美学**：采用玻璃态、渐变和微妙阴影

## 📐 组件原型

### 1. 顶部导航栏 (Header)

```

│ [🗺️ Dest. Wheel]                    [🌐 EN ▼] [⚙️] [☀️] │

设计规范：
- 高度: 64px
- 背景: 背景色 + 80% 透明度 + 毛玻璃效果 (backdrop-blur-md)
- 位置: 粘性定位 (sticky top-0)
- 阴影: 无 (使用边框分隔)
- Logo: MapPin 图标 + 文字组合
- 按钮: 幽灵按钮 (ghost variant)
```

#### 状态
- **默认状态**: 图标颜色 `--muted-foreground`
- **悬停状态**: 背景色 `--accent` 10% 透明度
- **激活状态**: 背景色 `--accent` 20% 透明度

## 🎬 交互流程

### 1. 抽取流程

```
用户点击 SPIN
      ↓
转盘开始旋转 (8圈 + 随机圈数)
      ↓
指针抖动效果
      ↓
转盘逐渐停止
      ↓
彩纸飘落动画 (confetti)
      ↓
获奖卡片滑入显示
      ↓
可选: 点击"进入市级抽签"
      ↓
面包屑更新 + 转盘重置
```

### 2. 收藏流程

```
用户点击心形按钮
      ↓
心形放大动画 (scale 1 → 1.3 → 1)
      ↓
心形填充红色
      ↓
保存到 localStorage
      ↓
可选: 打开设置面板查看收藏列表
```

### 3. 主题切换流程

```
用户点击主题按钮
      ↓
检测系统偏好或保存的主题
      ↓
切换主题类名 (light ↔ dark)
      ↓
平滑过渡动画 (0.3s)
      ↓
保存到 localStorage
```

## 🎨 动效规范

```javascript
// 动画配置
{
  duration: 6.5,  // 秒
  ease: [0.15, 0.85, 0.2, 1],  // 自定义缓动
  extraSpins: 8,

  transition: {
    type: "spring",
    stiffness: 600,

  }
```

```javascript
// 入场动画
{

    y: 20,
    scale: 0.95

  animate: {
    opacity: 1,
    y: 0,
    scale: 1
  },

    duration: 0.5,
    type: "spring"
  }
}
```

```javascript
// 悬停

// 过渡
```
