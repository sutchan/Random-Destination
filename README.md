# 随机目的地大转盘 v4.0.0 | 数据 v1.4.0

[English Version](./README_EN.md)

一个精美可自定义的旋转转盘，用于抽取随机旅游目的地，并提供 AI 驱动的见解、历史记录和收藏功能。支持"省 -> 市 -> 县"三级联动抽签。

## ✨ 功能特点

- **🎯 层级抽签**: 选定省份后可进入市级抽签，选定城市后可进入县级抽签。
- **🎨 精美UI设计**: 采用现代化设计，包括渐变色彩、精美的动画效果和视觉层次。
- **⚡ 流畅动画**: 使用 Motion 实现丝滑的动画效果，让每个交互都令人愉悦。
- **🎡 可自定义转盘**: 添加你自己的目的地或使用预设列表。
- **🤖 AI 见解**: 为抽中的目的地获取简要介绍和图片预览。
- **💖 历史与收藏**: 记录你的抽取历史并保存心仪的地点。
- **📱 响应式设计**: 针对手机、平板和桌面端进行了优化。
- **🌙 深色模式**: 支持系统主题和手动切换。
- **🚀 PWA 支持**: 渐进式 Web 应用，提供原生应用般的体验。

## 🛠️ 技术栈

- **框架**: Next.js 15+ (App Router)
- **样式**: Tailwind CSS 4
- **动画**: Motion (framer-motion)
- **AI**: Google Gemini 3 Flash
- **图标**: Lucide React
- **组件库**: shadcn/ui
- **主题**: 紫色渐变主题

## 🚀 快速开始

1. 克隆仓库。
2. 安装依赖: `npm install`。
3. 在 `.env` 中设置环境变量（可选）。
4. 启动开发服务器: `npm run dev`。

## 📸 设计亮点

- **现代配色方案**: 紫色主题色彩搭配和谐的渐变色转盘。
- **玻璃态效果**: 现代化的毛玻璃效果卡片设计。
- **微交互反馈**: 精致的按钮悬停和点击动画。
- **获奖体验**: 炫彩的获奖卡片展示和彩花效果。
