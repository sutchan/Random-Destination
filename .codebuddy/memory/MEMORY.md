# 项目长期记忆

## 项目信息
- 名称: Random Destination Wheel (random-destination)
- 版本: v3.5.0
- 描述: Next.js 随机旅游目的地转盘，支持省/市/县三级抽签 + AI 见解 + PWA
- 技术栈: Next.js 15+ (App Router), Tailwind CSS 4, Motion, shadcn/ui, Google Gemini

## 项目约定
- 代码注释使用中文
- 目录结构遵循 Next.js App Router 规范
- 组件: app/components/, Hooks: app/hooks/, 工具库: app/lib/, 国际化: app/locales/
- 提交格式: conventional commits

## 项目分支
- main: 唯一主分支，无 dev 分支
- 远程分支均已合并到 main

## 用户偏好
- 使用中文对话
- 输出精简，直接给结论
